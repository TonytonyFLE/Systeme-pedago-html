Mode d'emploi

Variante 1 : Travail en binômes avec la carte mentale

Distribuez une copie de la carte mentale à chaque binôme. Les apprenants utilisent le jeu de cartes pour classer les mots dans les différentes catégories en discutant de leurs choix personnels.

Variante 2 : Travail en grand groupe (recommandée)

Affichez la carte mentale au format A3. Utilisez le jeu de cartes numérique en mode projection. Chaque élève tire une carte à tour de rôle et explique dans quelle catégorie il la placerait et pourquoi.

Variante 3 : Jeu de conversation avec timer automatique

Utilisez le jeu de cartes avec le timer automatique. Dès qu'une nouvelle carte s'affiche, le timer de 3 minutes se lance. L'apprenant doit parler du mot affiché pendant toute la durée.

Utilisation du jeu de cartes :

• Cliquez sur "Nouvelle carte" pour afficher un mot aléatoire et démarrer le timer
• Utilisez "Démarrer/Pause" pour contrôler le timer
• L'apprenant peut passer à la carte suivante avant la fin du timer
• Le timer se remet automatiquement à 3 minutes pour chaque nouvelle carte

Questions pour lancer la discussion :

• "Dans quelle catégorie placez-vous ce mot ?"
• "Pourquoi ce choix ?"
• "Êtes-vous d'accord avec votre partenaire ?"
• "Pouvez-vous donner un exemple ?"
• "Et vous, qu'est-ce que vous en pensez ?"
